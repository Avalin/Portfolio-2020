Arachnophilia VR


Arachnophilia VR is a Unity project and is meant as a tool to alleviate arachnophobia and help the user to combat their fear with lifelike spider-interactions. The user controls how many spiders will be spawned and can squash, lift, throw the spiders to create an immersive and realistic experience.
