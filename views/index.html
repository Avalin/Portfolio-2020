Arachnophilia VR


Arachnophilia VR is a Unity project and is meant as a tool to alleviate arachnophobia and help the user to combat their fear with lifelike spider-interactions. The user controls how many spiders will be spawned and can squash, lift, throw the spiders to create an immersive and realistic experience. This project is part of my bachelor (ongoing), and I can show it to you during an interview.

2D Snake


2D Snake was made as part of an assignment during my Software Engineering bachelor. The purpose was for us to get comfortable with writing and using an update-loop and write the methods from scratch that usually game engines does for us. It is written in pure C# using Windows Forms to render the graphics.
Click here to check out the project.

T-Shirt Creator Kit


This project was made as a part of a lesson about shaders and UV-mapping on my sixth semester. You can select a t-shirt mesh, and give it any color and insert a logo on it. The different selectors will have animate separate parts of the shirt with Sine/Cos waves.
